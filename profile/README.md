# Praesidium
Praesidium is yet another password vault app, it aims to be simple and straightforward

## Features
- __Cross-platform__ -> Praesidium uses the target OS WebView achieving a uniform GUI and portability
- __Easy password management__ -> You can easily generate a strong password with a click, freeing you from thinking about a secure enough password

## Roadmap
- __Vault encryption__ -> Encrypt the user's vault so its useless even if compromised without the user's master password
- __OpenSSH key generation__ -> Allow the user to generate and store easily public/private key pairs for OpenSSH 

## Repositories
- [serde_cbored](https://github.com/PraesidiumApp/serde_cbored) -> Serves as the engine that allows Praesidium to read and write vaults
